# Barcode Modeling & Tracking

## Domain Concepts

Barcodes in inventory touch several distinct concerns:

- **Modeling** — what a barcode is (symbology + value + validation)
- **Assignment** — linking barcodes to variants (one variant, many barcodes)
- **Lookup** — resolving a scanned value back to a variant
- **Generation** — creating internal barcodes when suppliers don't provide them
- **Scan dispatch** — routing a scan to the correct workflow based on context

Each of these is a separate responsibility. Conflating them is the most common
mistake in barcode implementations.

// The symbology defines how the barcode value is structured and validated.

enum BarcodeSymbology: string
{
    case UPC_A    = 'upc_a';    // 12 digits — North American retail standard
    case UPC_E    = 'upc_e';    // 8 digits  — compact UPC
    case EAN_13   = 'ean_13';   // 13 digits — international retail standard
    case EAN_8    = 'ean_8';    // 8 digits  — compact EAN
    case CODE_128 = 'code_128'; // Alphanumeric — internal, logistics, shipping
    case QR       = 'qr';       // 2D — URLs, structured data, internal use
    case ITF_14   = 'itf_14';   // 14 digits — carton/case level (outer packaging)
    case GS1_128  = 'gs1_128';  // GS1-encoded logistics (SSCC, lot, expiry, etc.)

    public function label(): string
    {
        return match($this) {
            self::UPC_A    => 'UPC-A',
            self::UPC_E    => 'UPC-E',
            self::EAN_13   => 'EAN-13',
            self::EAN_8    => 'EAN-8',
            self::CODE_128 => 'Code 128',
            self::QR       => 'QR Code',
            self::ITF_14   => 'ITF-14',
            self::GS1_128  => 'GS1-128',
        };
    }
}

## 2. Barcode Value Object

```php
// Domain/Barcode/ValueObjects/Barcode.php
//
// Immutable. Validates format on construction so an invalid barcode
// can never exist in the domain.

final class Barcode
{
    public readonly string $value;

    public function __construct(
        public readonly BarcodeSymbology $symbology,
        string $rawValue,
    ) {
        $this->value = strtoupper(trim($rawValue));
        $this->validate();
    }

    public function equals(Barcode $other): bool
    {
        // Barcodes are globally unique by value regardless of symbology.
        // Two different symbologies should never share a value in practice.
        return $this->value === $other->value;
    }

    public function __toString(): string
    {
        return $this->value;
    }

    // -------------------------------------------------------------------------
    // Validation
    // -------------------------------------------------------------------------

    private function validate(): void
    {
        match($this->symbology) {
            BarcodeSymbology::UPC_A    => $this->validateUpcA(),
            BarcodeSymbology::EAN_13   => $this->validateEan13(),
            BarcodeSymbology::UPC_E    => $this->validateFixedLength(8),
            BarcodeSymbology::EAN_8    => $this->validateFixedLength(8),
            BarcodeSymbology::ITF_14   => $this->validateFixedLength(14),
            BarcodeSymbology::CODE_128 => $this->validateCode128(),
            BarcodeSymbology::QR       => $this->validateQr(),
            BarcodeSymbology::GS1_128  => $this->validateCode128(), // same charset
        };
    }

    private function validateUpcA(): void
    {
        if (!preg_match('/^\d{12}$/', $this->value)) {
            throw new \InvalidArgumentException("UPC-A must be exactly 12 digits: {$this->value}");
        }
        if (!$this->isValidGS1CheckDigit($this->value)) {
            throw new \InvalidArgumentException("UPC-A check digit is invalid: {$this->value}");
        }
    }

    private function validateEan13(): void
    {
        if (!preg_match('/^\d{13}$/', $this->value)) {
            throw new \InvalidArgumentException("EAN-13 must be exactly 13 digits: {$this->value}");
        }
        if (!$this->isValidGS1CheckDigit($this->value)) {
            throw new \InvalidArgumentException("EAN-13 check digit is invalid: {$this->value}");
        }
    }

    private function validateFixedLength(int $length): void
    {
        if (!preg_match("/^\d{{$length}}$/", $this->value)) {
            throw new \InvalidArgumentException(
                "{$this->symbology->label()} must be exactly {$length} digits: {$this->value}"
            );
        }
    }

    private function validateCode128(): void
    {
        // Code 128 supports ASCII 32–126. Reject control characters.
        if (empty($this->value) || !preg_match('/^[\x20-\x7E]+$/', $this->value)) {
            throw new \InvalidArgumentException(
                "Code 128 value contains invalid characters: {$this->value}"
            );
        }
    }

    private function validateQr(): void
    {
        if (empty($this->value)) {
            throw new \InvalidArgumentException('QR code value cannot be empty.');
        }
        // QR can encode virtually anything — length is the practical constraint.
        if (strlen($this->value) > 2953) {
            throw new \InvalidArgumentException('QR code value exceeds maximum length.');
        }
    }

    // -------------------------------------------------------------------------
    // GS1 Mod-10 Check Digit (used by UPC-A, EAN-13, ITF-14, EAN-8)
    // -------------------------------------------------------------------------

    private function isValidGS1CheckDigit(string $digits): bool
    {
        $sum = 0;
        $length = strlen($digits);

        // Check digit is the last digit; validate against the rest
        for ($i = 0; $i < $length - 1; $i++) {
            $digit = (int) $digits[$i];
            // Odd positions from the right (excluding check digit) get weight 3
            $weight = (($length - 1 - $i) % 2 === 1) ? 3 : 1;
            $sum += $digit * $weight;
        }

        $calculatedCheck = (10 - ($sum % 10)) % 10;
        $providedCheck   = (int) $digits[$length - 1];

        return $calculatedCheck === $providedCheck;
    }
}
```

## 4. VariantBarcodeSet Aggregate

```php
// Domain/Barcode/Aggregates/VariantBarcodeSet.php
//
// Owns all barcode assignments for a single variant.
// Enforces: only one primary barcode, no duplicate values.

class VariantBarcodeSet
{
    /** @var BarcodeAssignment[] */
    private array $assignments = [];

    private array $domainEvents = [];

    public function __construct(
        public readonly ProductVariantId $variantId,
    ) {}

    public function assign(
        Barcode $barcode,
        BarcodeSource $source,
        bool $makePrimary = false,
    ): BarcodeAssignment {
        // Guard: no duplicate barcode values within this set
        foreach ($this->assignments as $existing) {
            if ($existing->barcode->equals($barcode)) {
                throw new DuplicateBarcodeException(
                    "Barcode {$barcode->value} is already assigned to this variant."
                );
            }
        }

        // If makePrimary, demote any existing primary
        if ($makePrimary) {
            $this->assignments = array_map(
                fn(BarcodeAssignment $a) => $a->isPrimary
                    ? $this->cloneWithPrimary($a, false)
                    : $a,
                $this->assignments,
            );
        }

        // If this is the first assignment, make it primary automatically
        $shouldBePrimary = $makePrimary || empty($this->assignments);

        $assignment = new BarcodeAssignment(
            id: BarcodeAssignmentId::generate(),
            variantId: $this->variantId,
            barcode: $barcode,
            source: $source,
            isPrimary: $shouldBePrimary,
            assignedAt: new \DateTimeImmutable(),
        );

        $this->assignments[$assignment->id->value] = $assignment;

        $this->domainEvents[] = new BarcodeAssigned($this->variantId, $barcode);

        return $assignment;
    }

    public function revoke(BarcodeAssignmentId $assignmentId): void
    {
        $assignment = $this->assignments[$assignmentId->value] ?? null;

        if ($assignment === null) {
            throw new \DomainException("Assignment {$assignmentId->value} not found.");
        }

        if ($assignment->isPrimary && count($this->assignments) > 1) {
            throw new \DomainException(
                'Cannot revoke the primary barcode while other assignments exist. '
                . 'Promote another barcode to primary first.'
            );
        }

        unset($this->assignments[$assignmentId->value]);

        $this->domainEvents[] = new BarcodeRevoked($this->variantId, $assignment->barcode);
    }

    public function primaryBarcode(): ?BarcodeAssignment
    {
        foreach ($this->assignments as $assignment) {
            if ($assignment->isPrimary) {
                return $assignment;
            }
        }
        return null;
    }

    /** @return BarcodeAssignment[] */
    public function all(): array
    {
        return array_values($this->assignments);
    }

    public function releaseEvents(): array
    {
        $events = $this->domainEvents;
        $this->domainEvents = [];
        return $events;
    }

    private function cloneWithPrimary(BarcodeAssignment $a, bool $isPrimary): BarcodeAssignment
    {
        return new BarcodeAssignment(
            id: $a->id,
            variantId: $a->variantId,
            barcode: $a->barcode,
            source: $a->source,
            isPrimary: $isPrimary,
            assignedAt: $a->assignedAt,
        );
    }
}
```

## 5. BarcodeRegistry — The Lookup Service

```php
// Domain/Barcode/Services/BarcodeRegistry.php
//
// The single authoritative source for resolving a scanned barcode value
// to a ProductVariantId. Must enforce global uniqueness across all tenants
// (or per-tenant, depending on your multi-tenancy model — see note below).

class BarcodeRegistry
{
    public function __construct(
        private readonly BarcodeRepositoryInterface $repository,
    ) {}

    /**
     * Resolve a raw scanned value to a variant.
     * Throws if not found — the caller decides how to handle unknown barcodes.
     */
    public function resolve(string $scannedValue): ProductVariantId
    {
        $variantId = $this->repository->findVariantByBarcodeValue(
            strtoupper(trim($scannedValue))
        );

        if ($variantId === null) {
            throw new BarcodeNotFoundException(
                "No variant found for barcode: {$scannedValue}"
            );
        }

        return $variantId;
    }

    /**
     * Check if a barcode value is already registered (before assigning).
     * Used by the application layer before calling VariantBarcodeSet::assign().
     */
    public function isRegistered(string $value): bool
    {
        return $this->repository->findVariantByBarcodeValue(
            strtoupper(trim($value))
        ) !== null;
    }
}
```

## 6. InternalBarcodeGenerator

```php
// Domain/Barcode/Services/InternalBarcodeGenerator.php
//
// Generates a Code 128 barcode for variants that don't have a supplier UPC.
// Format: {PREFIX}-{TENANT_SHORT}-{VARIANT_ID_FRAGMENT}
// Example: INV-T42-0F3A9B
//
// Alternatively: if the merchant has a GS1 company prefix, you can generate
// compliant EAN-13s. That's a separate (more complex) path.

class InternalBarcodeGenerator
{
    private const PREFIX = 'INV';

    public function __construct(
        private readonly BarcodeRegistry $registry,
    ) {}

    public function generate(ProductVariantId $variantId, TenantId $tenantId): Barcode
    {
        // Build a deterministic but opaque value from the variant ID
        // Retry on the (very unlikely) collision with an existing barcode
        $attempts = 0;

        do {
            $value = $this->buildValue($variantId, $tenantId, $attempts);
            $attempts++;

            if ($attempts > 5) {
                throw new \RuntimeException('Could not generate a unique barcode after 5 attempts.');
            }
        } while ($this->registry->isRegistered($value));

        return new Barcode(BarcodeSymbology::CODE_128, $value);
    }

    private function buildValue(ProductVariantId $variantId, TenantId $tenantId, int $salt): string
    {
        $tenantFragment  = strtoupper(substr(md5($tenantId->value), 0, 4));
        $variantFragment = strtoupper(substr(md5($variantId->value . $salt), 0, 8));

        return self::PREFIX . '-' . $tenantFragment . '-' . $variantFragment;
        // e.g. INV-A3F2-0C8E4B1D
    }
}
```

## 7. BarcodeScanDispatcher — Context-Aware Routing

```php
// Domain/Barcode/Services/BarcodeScanDispatcher.php
//
// A scan is just a value — what happens next depends entirely on the workflow
// context the scan occurs in. This dispatcher makes that routing explicit
// rather than leaving it scattered across UI controllers.

enum ScanContext: string
{
    case PointOfSale  = 'pos';
    case Receiving    = 'receiving';
    case CycleCount   = 'cycle_count';
    case TransferOut  = 'transfer_out';
    case TransferIn   = 'transfer_in';
}

// Each context implements this interface
interface ScanHandlerInterface
{
    public function handle(ProductVariantId $variantId, string $rawScan, array $context): void;
}

class BarcodeScanDispatcher
{
    /** @var ScanHandlerInterface[] keyed by ScanContext value */
    private array $handlers = [];

    public function __construct(private readonly BarcodeRegistry $registry) {}

    public function register(ScanContext $context, ScanHandlerInterface $handler): void
    {
        $this->handlers[$context->value] = $handler;
    }

    public function dispatch(string $rawScan, ScanContext $context, array $payload = []): void
    {
        // Step 1: resolve the scanned value to a variant
        // Throws BarcodeNotFoundException if not recognized
        $variantId = $this->registry->resolve($rawScan);

        // Step 2: route to the correct handler for this workflow context
        $handler = $this->handlers[$context->value] ?? null;

        if ($handler === null) {
            throw new \DomainException("No handler registered for scan context: {$context->value}");
        }

        $handler->handle($variantId, $rawScan, $payload);
    }
}
```

```php
// Example handler — POS scan triggers inventory decrement
class PosScanHandler implements ScanHandlerInterface
{
    public function __construct(
        private readonly InventoryService $inventoryService,
    ) {}

    public function handle(ProductVariantId $variantId, string $rawScan, array $context): void
    {
        $this->inventoryService->decrementForSale(
            variantId: $variantId,
            quantity: $context['quantity'] ?? 1,
            saleId: $context['saleId'],
            actor: ActorId::from($context['actorId']),
        );
    }
}
```

Schema::create('barcode_assignments', function (Blueprint $table) {

    $table->timestamps();

    // Global uniqueness — two tenants cannot share a barcode value.
    // If you want per-tenant uniqueness instead, change this to:
    // $table->unique(['tenant_id', 'barcode_value']);
    $table->unique('barcode_value');

    $table->foreign('variant_id')->references('id')->on('product_variants');
});

## Key Design Decisions

| Decision | Rationale |
|---|---|
| Check digit validation in the value object | An invalid UPC can never enter the domain — catches data entry errors at the boundary |
| Global barcode uniqueness (not per-tenant) | Real UPCs are globally unique; two tenants scanning the same product should resolve to their own variant, handled at the registry layer via tenant context |
| `VariantBarcodeSet` as the aggregate | Keeps the one-primary invariant and duplicate-value guard together in one place |
| `BarcodeRegistry` as a separate service | Lookup is a cross-cutting concern that doesn't belong on the variant or the set |
| `BarcodeScanDispatcher` | Keeps scan routing explicit and testable; adding a new scan context is adding a new handler, not modifying existing code |
| `isPrimary` flag | Variants often have both a supplier UPC and an internal barcode; the primary determines what gets printed on shelf labels |

## Multi-Tenancy Note

Global barcode uniqueness is correct for supplier-issued barcodes (UPCs are globally
registered). For internal barcodes, you have two options:

**Option A — Global uniqueness (simpler):** The `InternalBarcodeGenerator` encodes
the tenant ID into the barcode value, making collisions impossible by construction.

**Option B — Per-tenant uniqueness:** Change the unique index to
`(tenant_id, barcode_value)`. The registry then always scopes lookups by tenant.
This allows two tenants to have the same internal code, but requires the scan
dispatch context to always carry a `tenantId`.

Option A is recommended unless you have a specific reason to allow shared values.
